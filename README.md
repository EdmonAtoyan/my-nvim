# My Neovim IDE

VS Code-like Neovim configuration with:
- Material Theme & Icons
- Integrated Terminal
- Auto-completion (LSP)
- Works on GUI and headless Linux servers

## Installation

```bash
git clone https://github.com/USERNAME/my-nvim.git](https://github.com/EdmonAtoyan/my-nvim.git)
ln -s my-nvim/nvim ~/.config/nvim
nvim
